# Upscaler Hub

An open-source tool for managing resolution upscaling in Windows games.

Game developers often ship titles with outdated DLSS, FSR, or XeSS files. Upscaler Hub finds your installed games and lets you swap those files with newer versions. You can also inject OptiScaler wrappers and force driver-level rendering presets.

## Features

- **Native Swapper:** Replace DLSS, FSR, or XeSS `.dll` files without digging through game folders.
- **OptiScaler Injection:** Add resolution scaling wrappers to DirectX 11/12, Vulkan, and OpenGL games.
- **NVAPI Presets:** Override default DLSS rendering presets using Nvidia's driver API.
- **Auto-Scanner:** Find games installed via Steam, Epic Games, or local directories.
- **Localization:** English and Brazilian Portuguese.

## Installation

Download the latest `UpscalerHub.exe` from the [Releases](https://github.com/YuukiFST/UpscalerHub/releases) page. The executable runs standalone. No installation required.
